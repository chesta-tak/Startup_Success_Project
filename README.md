# 🚀 Startup Success Prediction Project

An AI-powered web application that predicts the probability of a startup’s success using real Indian funding data (2020–2025).



## 📌 Project Overview
StartupLab helps entrepreneurs and analysts evaluate startup success chances based on:
- Industry
- City
- Funding amount
- Business stage

The system uses a Machine Learning model integrated with a Flask backend and a responsive frontend UI.

## ▶️ How to Run the Project

### 1️⃣ Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
---
###2️⃣ Frontend Setup
```bash
cd frontend
python -m http.server 5500
```
